# Nexus: Venue Management Intelligence
Nexus is a state-of-the-art Venue Management System designed to improve the physical event experience at large-scale sporting venues. Build with a focus on real-time crowd movement tracking, wait time monitoring, and rapid staff coordination.
## 🚀 Features
- **Live Overview Dashboard**: Instantly track global metrics including total attendance, average queue wait times, and active operating staff.
- **Advanced Heatmap Intelligence**: Interactive venue zones showing live congestion statuses (Flowing, Busy, Congested) using a dynamic color-coded system to identify bottlenecks.
- **Incident & Alert Management**: Automated tracking of bottlenecks or logistical issues (e.g., low supplies in restrooms) with real-time response options to dispatch staff instantly.
- **Staff Roster Hub**: Visualize all deployed units and their current status, with options to reassign staff or broadcast global dispatch commands.
- **Real-Time Notification System**: Integrated toast notifications provide immediate feedback on critical actions or dispatched units.
## 🛠️ Technology Stack
Built utilizing modern, functional web tools:
- **Core Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: Custom modern CSS Architecture with responsive Glassmorphism design and micro-animations.
- **Iconography**: [Lucide React](https://lucide.dev/) for premium crisp dashboard vector icons.
- **Typography**: [Inter](https://fonts.google.com/specimen/Inter) and [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts.
## 🖥️ Local Setup & Installation
If you would like to run this application locally, you will need [Node.js](https://nodejs.org/) installed on your machine.
1. **Clone or download the repository**, and navigate into the project directory:
   ```bash
   cd antigrav_proj
   ```
2. **Install the dependencies**:
   ```bash
   npm install
   ```
3. **Start the local development server**:
   ```bash
   npm run dev
   ```
4. **View the Application**:  
   Open your browser and navigate to the local URL provided by Vite (typically `http://localhost:5173`).
## 🎨 Design Philosophy
The system prioritizes visual excellence and readability under pressure. By using deep dark-mode tones (`#050508`) paired with bright contextual highlights (Emerald, Amber, Rose), operators can spot logistical friction in milliseconds without the typical "cognitive overload" associated with legacy administrative software.
